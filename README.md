MealFinder is a recipe discovery app built in Swift using SwiftUI and MVVM architecture.  
It lets users browse and search meals via TheMealDB API, save their favorites locally with SwiftData, and uses Gemini AI to parse raw recipe instructions into clean, structured steps.
